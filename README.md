# 🏦 AI-Powered Loan Approval & Credit Risk Assessment System

## 📌 Project Overview

The AI-Powered Loan Approval & Credit Risk Assessment System is a machine learning-based web application developed using Python, Scikit-Learn, Streamlit, Plotly, and SQLite.

The system predicts loan approval eligibility based on applicant information and provides risk assessment, approval probability, business insights, analytics dashboards, and PDF report generation.
